### xlsx-to-clickup

This project create tasks of the xlsx(Excel) to http://app.clickup.com

#### Configuration
This .env file
```
CLICKUP_API_TOKEN=myToken
LIST_ID=codeYourListAddTasks
FILE=./tmp/example.xlsx
STATUS=BACKLOG
TAGS=backend,update,devops
```

#### Execute project
`yarn run xlsxToClickup`
